# Amazon Clone 🛒

A responsive front-end clone of Amazon's e-commerce homepage built using HTML & CSS. 
This project replicates Amazon's UI/UX with modern web design principles.

## ✨ Features
- **Responsive Design**: Works seamlessly on Mobile, Tablet & Desktop
- **Navigation Bar**: Amazon-style header with logo, search bar, cart icon
- **Product Listing**: Grid layout for product cards with images, price, ratings
- **Interactive Elements**: Hover effects on buttons, cards, and links
- **Search UI**: Functional search bar design
- **Footer**: Multi-column footer like original Amazon

## 🛠️ Tech Stack
- **HTML5**: Semantic markup for structure
- **CSS3**: Flexbox, Grid, Media Queries for responsiveness
- **Google Fonts & Icons**: For Amazon-like typography
